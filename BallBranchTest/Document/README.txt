Ball Branch Test Program

How to run:
1. Open Visual Studio 2017 project : BallBranchTest.sln
   Press F5 to execute.
2. or You can execute "BallBranchTest\BallBranchTest\bin\Debug\BallBranchTest.exe"
3. User needs to input depth of tree. Default depth is 4.
4. The program will   
   (1) show the initial tree gates status
   (2) show prediction result
   (3) show simulation

5. The document folder has an example of result (result1.txt).


Time of solution: about 2 hours
1. design the solution : 0.5 hour
2. implement the code : 1 hour
3. debugging and document : 0.5 hour
